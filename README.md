# Typing-with-gaze
Type letters on blinking eyes when they are active and then write it to file. 
Gives better experience with extra blinking effects for keyboard and a loading status while selecting a letter. 
Includes alphabets,numbers and space. Fixed bugs while the playing sounds that you can find in other similar projects.
Includes common python modules(OpenCV,dlib and pygame).Please install them if you don't have. Just extract and run.
